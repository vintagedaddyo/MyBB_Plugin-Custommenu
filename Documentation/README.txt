MyBB-Plugin-Custommenu

A plugin for MyBB 1.8.x to easily manage the forums main menu

Requires the PluginLibrary plugin *(included in pkg)


Plugin Installation:

1) Upload the contents of the "Upload" directory into your forums root directory

2) Make sure the contained PluginLibrary Plugin is installed

3) Visit: ACP »  Home » Plugins and install the Custom Menu Plugin

4) Visit: ACP »  Home » Board Settings » Custom Menus link in the side menu to access the  Home » Custom Menus » List Menus
